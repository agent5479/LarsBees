Site Management

Track multiple apiary sites with GPS coordinates, site classifications, and detailed site information. Interactive maps and site grouping for efficient management.

Key Features:
  • GPS Integration: Precise location tracking with Google Maps integration
  • Site Classifications: Functional (Production, Nucleus, Queen Rearing) and Seasonal (Summer/Winter) classifications
  • Hive Inventory: Track individual hives with strength monitoring and stack configurations
  • Landowner Management: Store contact information and access permissions
  • Visual Hive Grid: Interactive grid system for hive layout management
  • Archive System: Hide inactive sites while preserving data

Performance Analytics

Advanced reporting and analytics with real-time data visualization. Track hive strength, productivity, health metrics, and operational efficiency.

Analytics Features:
  • Hive Performance Charts: Visual tracking of hive strength over time
  • Health & Mortality Reports: Disease tracking and mortality analysis
  • Operations Summary: Task completion rates and efficiency metrics
  • Data Export: CSV export for external analysis and reporting
  • Real-time Updates: Live data synchronization across all reports
  • Mobile Responsive: Optimized charts for mobile and desktop viewing

Task Scheduling

Comprehensive task management with scheduling, reminders, and progress tracking. Seasonal requirements and automated task suggestions.

Scheduling Capabilities:
  • Task Categories: Inspection, Feeding, Treatment, Harvest, Maintenance, and more
  • Priority Levels: Normal, High, and Urgent task prioritization
  • Due Date Tracking: Visual calendar with overdue indicators
  • Bulk Operations: Schedule multiple tasks across sites efficiently
  • Task Templates: Pre-configured task sets for common operations
  • Progress Tracking: Mark tasks complete with timestamps and notes

Team Management

Multi-user support with role-based permissions. Employee management, task assignment, and collaborative workflow management.

Team Features:
  • Role-Based Access: Admin and Employee roles with appropriate permissions
  • Employee Activation: Secure activation system with temporary passwords
  • Tenant Isolation: Multi-tenant architecture for data security
  • Permission Control: Employees can view/update but not delete critical data
  • Activity Tracking: Monitor who made changes and when
  • Secure Login: No hardcoded passwords - all credentials stored securely

Real-time Sync

Firebase-powered real-time data synchronization across all devices. Offline support with automatic sync when connection is restored.

Sync Features:
  • Persistent Sync Status: Real-time overlay showing sync status and pending changes
  • Offline Support: Continue working without internet - changes queue locally
  • Automatic Reconnection: Queued changes sync when connection restored
  • Data Integrity: Conflict resolution and data validation
  • Multi-Device Sync: Changes appear instantly across all devices
  • Change Tracking: Detailed logging of all data modifications

Data Security

Enterprise-grade security with encrypted data transmission and storage. Multi-tenant architecture for data isolation and privacy protection.

Security Features:
  • No Hardcoded Passwords: All credentials stored securely in GitHub Secrets
  • Firebase Security: Encrypted data transmission and storage
  • Multi-Tenant Architecture: Complete data isolation between organizations
  • Role-Based Permissions: Granular access control for different user types
  • Secure Authentication: Temporary passwords with device remembering
  • Open Source: Transparent, auditable codebase for security review
